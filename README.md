# 🏠 FitMySpace

**FitMySpace** is a modern, high-performance search engine designed to help users find the perfect home organization products across various e-commerce platforms.

Inspired by the clean and aesthetic approach of _The Home Edit_, this project combines a minimalist design with powerful filtering capabilities, allowing users to organize their homes room by room with ease.

## ✨ Key Features

- **Smart Search:** Find organizers across multiple stores in one place.
- **Room-Based Filtering:** Tailored results for Kitchen, Pantry, Bathroom, Closet, and more.
- **Accessible by Design:** Built from the ground up using Radix UI Primitives to meet WCAG standards.
- **The Home Edit Aesthetic:** A clean, clutter-free UI with subtle rainbow accents.
- **Fast & Reliable:** Powered by Next.js SSR/ISR and React Query for a seamless experience.

## 🛠️ The Tech Stack

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** TailwindCSS 4 (Experimental/Next-gen)
- **Components:** Radix UI Primitives
- **Data Fetching:** TanStack Query (React Query)
- **Testing:** Jest, React Testing Library & MSW
